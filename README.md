# Agricultural Adaptation Analysis

This repository contains an academic business intelligence and predictive analytics project focused on assessing agricultural adaptation needs under environmental and economic stressors.

## Project Overview

This project was completed as an academic analysis of how agricultural, environmental, and economic indicators can be used to predict whether crop adaptation measures are required. The analysis considers factors such as precipitation, temperature, soil type, water stress, crop yield, market access, technology access, and extreme weather events.

The project applies business intelligence methods and machine learning techniques to support data-driven resilience planning in agriculture. The final report presents the business problem, dataset design, exploratory analysis, interdependence analysis, dependence analysis, model comparison, and strategic implications for agricultural adaptation planning.

## Academic Context

This repository was created for academic and portfolio purposes. The analysis, report, datasets, and supporting files are intended to demonstrate applied business intelligence, statistical analysis, and predictive modeling techniques in an agricultural resilience context.

The project should not be interpreted as a production-ready decision system or as professional agricultural, environmental, financial, or policy advice.

## Methods Used

The project uses the following analytical methods:

- Exploratory data analysis
- Principal component analysis
- Hierarchical clustering
- K-means clustering
- Logistic regression
- Decision tree analysis
- Neural networks
- Support vector machines
- Ensemble modeling
- ROC, AUC, and lift curve evaluation

## Tools Used

- JMP
- Microsoft Excel
- Business intelligence analysis
- Predictive modeling
- Data visualization

## Software Requirement

The JMP data files (`.jmp`) and JMP journal files (`.jrn`) included in this repository require JMP software to open, explore, or modify. Users must have JMP installed on their device and must have access to a valid JMP license.

Users without JMP may still review the final PDF report and Excel workbooks to understand the project methodology, results, and model evaluation outputs.

## Key Objective

The objective of this academic project is to identify whether agricultural regions require adaptation based on environmental, economic, and agricultural indicators. The analysis is intended to demonstrate how predictive analytics and business intelligence methods can support proactive resource allocation, risk mitigation, and resilience planning for agricultural stakeholders.
